10 July 2018

TheLion1675 updated:

  • Added more jobs to the list, they should automatically appear. All jobs are now functional, but may not have spawnpoints yet.
  • All jobs now start with a handheld radio.
  • Added CE's office in Engineering.
  • Removed Brig Officer for the time being. This role is being evaluated.
  • Removed atmos control for upper EZ for the time being, while the atmos system is being sorted.
  • Removed a lot of redundant code regarding jobs and outfits. Should help with compile time some.
  • Gate B now requires level 4 Security clearance.
  • Fixed a bug with SCP-151 being visible through shutters.
  • Fixed an issue with Atmos in LCZ.
  • Fixed a bug with Servers windoor in Engineering.
  • Fix to Keycard auth devices. It now requires two level 4 Security clearance keycards to detonate.
  • Gate A can finally be opened.
  • Fixed an issue with D-Class cells not being powered properly.
  • Fixed RnD labs not being hooked up to power net.
  • Fixed a scrubber not having been mapped in near RnD.
  • Fixed Sergeant ranks not having the correct shorthand.
  • Fixed an issue with d-cells not opening.
  • In order to fix the above point I had to add basic Security access requirements to all cell doors, but the master switch remains in the airlock oversight.
  • Minor fixes in Engineering.
  • Mapped in Engineer spawns.
  • Added in a replacement sound for Delta alert.
  • Added in a replacement sound for the power down random event.
  • Added in a replacement sound for power up random event.
  • Disabled Communications Officer from spawning while this job is being figured out behind the scenes. Manning just the comms station can be boring, right?
